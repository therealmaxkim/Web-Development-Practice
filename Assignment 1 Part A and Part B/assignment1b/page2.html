Relative Sort Array

Problem Description

Given two arrays arr1 and arr2, the elements of arr2 are distinct, and all elements in arr2 are also in arr1.

Sort the elements of arr1 such that the relative ordering of items in arr1 are the same as in arr2. Elements that don't appear in arr2 should be placed at the end of arr1 in ascending order.

Examples

Input:

arr1 = [2,3,1,3,2,4,6,7,9,2,19], arr2 = [2,1,4,3,9,6]

Output:

[2,2,2,1,4,3,3,9,6,7,19]

Constraints

  • arr1.length, arr2.length <= 1000
  • 0 <= arr1[i], arr2[i] <= 1000
  • Each arr2[i] is distinct.
  • Each arr2[i] is in arr1.

Questions

  1. Can arr1 be empty?

Observations

  1. arr1 is not distinct. but arr2 is distinct
  2. both arrays are randomly ordered
  3. there may be or not be repeated numbers in arr1
  4. arr1 has numbers from arr2 or not

Ideas

Idea 1

  1. part 1: sort arr1 that are in arr2
  2. part 2: sort arr1 that are not in arr2

you only need to remember how many of one number exists. You can sort them into a list later.

  1. First, map the numbers in arr2 with the indices.
  2. Then, have another map that maps each number with the frequency that htey appear
  3. if a number in arr1 doesn't appear in arr2, simply add it to another list
  4. have a new list and append each number in arr2 by their order and append a number of times equal to frequency.
