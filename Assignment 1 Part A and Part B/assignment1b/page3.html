Remove Nth Node From End of List

Problem Description

Given a linked list, remove the n-th node from the end of list and return its head.

Examples

Input:

Given linked list: 1->2->3->4->5, and n = 2.

After removing the second node from the end, the linked list becomes 1->2->3->5.

Output:

1->2->3->5

Note:

Given n will always be valid.

Followup:

Could you do this in one pass?

Questions

  1. Can n be zero?

Observations

  1. n is always valid, and can be from n to length of list-1
  2. if n is zero, then you are deleting the head of the list
  3. is removing nth from the end equivalent to something else?

Ideas

Idea 1: Two Pass Method

  1. Find the length of the list (traverse once with a counter until pointer is None)
  2. Traverse to node and delete it (traverse to the node before that needs to be deleted)
  3. Return the head
  4. If n is equal to the length, simply return the new head (head.next)

Edge cases: deleting the first element of the list

Idea 2: One Pass Method

  1. Save all references into a list with indices corresponding to order
  2. Lookup the reference to node and delete it

Correct idea: One Pass Method (two pointer)

  1. Initialize a pointer, A at first node
  2. Initialize a pointer, B at n nodes ahead
  3. Increment both pointers until B's next is None
  4. At that point, A will be pointing to the node right before the node that needs to be deleted
  5. Return the head's next if you are trying to delete the head

edge case: if B goes out of bounds trying to go n nodes ahead, then you are trying to delete the first node
